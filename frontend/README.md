# AgentStudio Frontend

这是 AgentStudio 平台的前端项目，基于 Next.js 16 + TypeScript + Tailwind CSS 构建。

## 快速启动

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

浏览器访问 [http://localhost:3000](http://localhost:3000) 即可查看页面。
